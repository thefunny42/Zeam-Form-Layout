================
zeam.form.layout
================

``zeam.form.layout`` let you embbed a `zeam.form.base`_ form inside a
layout created with `grokcore.layout`_.

You just have to import ``Form`` (or ``ComposedForm``) from
``zeam.form.layout`` instead of their original package.

Default templates are defined ready to be used inside a layout.

.. _zeam.form.base: http://pypi.python.org/pypi/zeam.form.base
.. _grokcore.layout: http://pypi.python.org/pypi/grokcore.layout
